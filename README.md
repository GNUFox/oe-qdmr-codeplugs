# OE QDMR Codeplugs

A Repository for DMR-Codeplugs for Austria (and neighbouring countries / regions).  
Using YAML Format readible by qdmr (https://dm3mat.darc.de/qdmr/).

**WARNING: There may be some errors. Work in Progress**
